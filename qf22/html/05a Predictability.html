Return predictability¶

Start with the definition of gross return,

\[1+R_{t+1} = \frac{P_{t+1} + D_{t+1}}{P_t},\]

and rearrange so all the variables about the future are on the right-hand side,

\[P_t = \frac{P_{t+1} + D_{t+1}}{1+R_{t+1}}.\]

Suppose that the expected return is constant, so \(\E(R_{t+1}) = R\). The we can take expectations of both sides of the equation to get

\[P_t = \E\left(\frac{P_{t+1} + D_{t+1}}{1+R_{t+1}}\right) = \frac{\E(P_{t+1}+D_{t+1})}{1+R},\]

which we can also write as

(10)¶\[P_t = \frac{\E(D_{t+1})}{1+R} + \frac{\E(P_{t+1})}{1+R}.\]

The same equation holds in any period, so, moving forward one period, we can write

\[P_{t+1} = \frac{\E(D_{t+2})}{1+R} + \frac{\E(P_{t+2})}{1+R}.\]

Substituting this into (10), we get

\[P_t = \frac{\E(D_{t+1})}{1+R} + \frac{\E(D_{t+2})}{(1+R)^2} + \frac{\E(P_{t+2})}{(1+R)^2}.\]

We can continue to iterate forward this equation \(K\) periods, giving

\[P_t = \sum_{k=1}^K \frac{\E(D_{t+k})}{(1+R)^k} + \frac{\E(P_{t+K})}{(1+R)^K}.\]

The assumption that \(\lim_{K\rightarrow\infty} \frac{\E(P_{t+K})}{(1+R)^K} = 0\) is known as the “no bubbles” condition or by its fancier technical name of a transversality condition.

If we let \(K \rightarrow \infty\) and assume that the second term goes to zero, we have the dividend discount model:

\[P_t = \sum_{k=1}^{\infty} \frac{\E(D_{t+k})}{(1+R)^k}.\]

This is often written as

\[P_t = \E_t \left( \sum_{k=1}^{\infty} \frac{D_{t+k}}{(1+R)^k} \right)\]

where the subscript \(t\) on the expectations operator emphasizes that we are taking expectations using information that is available at that time.

Shiller [1981] points out that if this equation holds, then the variance of price (on the left hand side) should equal the variance of the right hand side — meaning realized dividends plus some statistical noise. In other words, prices should be less volatile than future realized dividends. Empirically, he showed convincingly that this is not the case.

If we assume further that dividends grow at a constant rate \(G\), so

\[\E(D_{t+k}) = (1+G)^{k-1} \E(D_{t+1}),\]

then the dividend discount model simplifies to the famous Gordon growth model:

\[P_t = \frac{\E(D_{t+1})}{R-G}.\]

Equivalently,

\[R = \frac{D}{P} + G,\]

which says simply that returns are generated by either income (dividends) or capital gains (which must equal the growth rate of dividends over the long run).

Total return index¶

Suppose that in each period the investor reinvests all dividends by buying new shares at the prevailing price. Starting with \(N_0\) shares at time \(t=0\), the number of shares will increase over time following

\[N_{t+1} = N_t \left(1+\frac{D_{t+1}}{P_{t+1}}\right).\]

Shiller’s dividend series is expressed as an annual number (despite being reported monthly), so to calculate the monthly change in shares, we divide dividends by 12.

T = len(shiller)

Nt = np.zeros(T)
Nt[0] = 1
for t in range(1, T):

Filtering a series¶

An alternative approach to generating a series like \(N_t\) is to use a linear filter. Given an input series \(x_t\), the output \(y_t\) of a linear filter is defined as

\[\begin{align*} a_0 y_t = b_0 x_t &+ b_1 x_{t-1} + \cdots + b_M x_{t-M} \\ & - a_1 y_{t-1} - \cdots + a_N y_{t-N}. \end{align*}\]

This is implemented in scipy using the lfilter function

lfilter(b, a, x)

where b, a, and x are all vectors.

For example, to generate values for \(y\) following

\[y_t = 0.5 + 0.1 y_{t-1}\]

we pass the following parameters:

from scipy import signal

signal.lfilter([1], [1, -0.1], [0.5, 0.5, 0.5, 0.5])

array([0.5   , 0.55  , 0.555 , 0.5555])

To use a linear filter to create our nshares series, we simply take logs,

\[n_{t+1} = n_t + \log\left(1+\frac{D_{t+1}}{P_{t+1}}\right).\]

avg47 = pd.Series(shiller.loc['1947':,'CAPE'].mean(), index=shiller.loc['1947':].index)
curentval = pd.Series(shiller['CAPE'].iloc[-1], index=shiller.index)

# main plot
ax = shiller['CAPE'].plot(figsize=(9,6), ylim=(0,45), legend=True, x_compat=True, c='b', lw=1)
