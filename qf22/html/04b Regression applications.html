Implementing regressions¶

import numpy as np
import matplotlib as mpl
import matplotlib.pyplot as plt

# Regenerate model from previous section
from numpy.random import default_rng
rng = default_rng(8675309)
N = 100
x = rng.uniform(high=10, size=(N,1))
e = rng.normal(0, 7, size=(N,1))
y = 2 - 3*x + e

Standard errors and t-statistics¶

sklearn doesn’t give standard errors but we can calculate them.

The variance-covariance matrix of the coeffient estimates is given by \(\hat\sigma^2(X'X)^{-1}.\)

from numpy.linalg import inv

RSS = (1 - lin_reg.score(x,y)) * y.var() * N

N, K = x.shape

varcov = RSS/(N-K) * inv(X.T @ X)

varcov

array([[ 1.92488865, -0.28082839],
       [-0.28082839,  0.05095225]])

The diagonal elements of this matrix give the estimated variance of each coefficent, which we can use to calculate the standard errors.

np.sqrt(varcov.diagonal())

array([1.38740356, 0.22572605])

np.c_[(lin_reg.intercept_, lin_reg.coef_)] / np.sqrt(varcov.diagonal())

array([[  1.62177867, -14.23433632]])

Nonlinear data¶

What if our data isn’t linear?

x = 6 * rng.uniform(size=(n,1)) - 3
y = 2 + x + 0.5*x**2 + rng.standard_normal(size=(n, 1))

plt.plot(x, y, "bo")
plt.xlabel('x', fontsize=16)
plt.ylabel('y', rotation=0, fontsize=16)
plt.axis([-3, 3, 0, 10])
plt.show()

X = np.c_[np.ones(n), x, x**2]
X[:5]

model2 = sm.OLS(y, X)
